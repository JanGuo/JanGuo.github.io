Linux安装java1.8

  • 下载javaSE安装包

https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

  • 使用FileZilla将java压缩文件上传到服务器

  • 使用Xshell与服务器进行连接

以上两个步骤需要通过ssh与服务器进行连接,因此服务器上要安装ssh服务

sudo apt-get update

安装ssh服务—sudo apt-get install openssh-server

查看ssh服务是否启动— sudo ps -e |grep ssh

启动ssh服务—sudo service ssh start

在连接的过程中可能会出现一些问题比如:

  • 服务器拒绝了密码

  • sshd的设置不允许root用户远程登录

    sudo vim /etc/sshd/ssh_config

然后reboot或者service sshd restart即可

vim 修改文件出现错误“E45: ‘readonly’ option is set (add ! to override)”

如果是root权限，可以
:wq! 强行保存退出

将java1.8.tar.gz文件和tomcat.tar.gz上传到服务器

使用tar -zxvf jdk..tar.gz将文件解压之后配置jdk1.8的环境变量

使用命令打开/etc/profile文件

sudo vi /etc/profile
在末尾添加以下几行文字：

export JAVA_HOME=/usr/local/jdk1.8.0_211
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH

执行命令使修改立即生效

source /etc/profile

或者重启服务器(我用的是虚拟机可以使用reboot命令)

使用java -version 查看配置的环境变量是否生效
