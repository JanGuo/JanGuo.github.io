虚拟机Ubantu18.04设置静态IP

发表于 2019-07-28

Ubantu18.04设置固定IP

ubuntu18与ubuntu14、16设置静态ip地方方法不同，很多人没去读更新文档的时候往往会设置静态ip地址不成功，下面是具体的设置方法

查看ip的两个命令

ifconfig
ip route show

配置本机网络适配器

参考文章 https://blog.csdn.net/u014466635/article/details/80284792

  1. VM->编辑->虚拟网络编辑器

  2. 设置子网IP 子网掩码

  3. 设置NAT网关

  4. 修改适配器

修改虚拟机的操作

1.做之前一定要确认自己操作系统的版本，每个版本设置的方法有时完全不相同

2.vim /etc/netplan/*.yaml

yaml文件名称不同，这里以正则匹配

3.修改之前的配置文件是这样的，默认是dhcp获取ip地址

network:
        ethernets:
            ens33:
                dhcp4: ture
        version: 2

4.修改后的内容如下图，保存并退出

# This file is generated from information provided by
# the datasource.  Changes to it will not persist across an instance.
# To disable cloud-init's network configuration capabilities, write a file
# /etc/cloud/cloud.cfg.d/99-disable-network-config.cfg with the following:
# network: {config: disabled}
network:
    ethernets:
        ens33:
            addresses: [192.168.174.132/24]
            gateway4: 192.168.174.2
            dhcp4: no
    version: 2

5.sudo netplan apply

执行完成终端会掉线，因为ip地址已经更换了

6.这个时候域名是不能访问的，需要配置DNS

7.手动指定DNS

参考文章 https://www.cnblogs.com/breezey/p/9155988.html

vim /etc/systemd/resolved.conf

[Resolve]
DNS=114.114.114.114
#FallbackDNS=
#Domains=
LLMNR=no
#MulticastDNS=no
#DNSSEC=no
#Cache=yes
#DNSStubListener=yes

8.重启
